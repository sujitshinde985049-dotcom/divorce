# Divorce Company Website

A modern, responsive 5-page website for a divorce and family law company. The website is mobile-friendly and includes a contact form.

## Features

* Responsive mobile-friendly design
* Modern and professional UI
* 5 Pages:

  * Home
  * About Us
  * Services
  * Testimonials
  * Contact Us
* Contact form
* Easy to customize


## Getting Started

### 1. Download the Project

Clone the repository:

```bash
git clone https://github.com/your-username/divorce-company-website.git
```

### 2. Open Locally

Open `index.html` directly in your browser or use a local server.

### 3. Customize

Update the following information:

* Company Name
* Logo
* Phone Number
* Email Address
* Office Address
* Services and Content

## Deploy on Vercel

1. Push the project to GitHub.
2. Sign in to https://vercel.com.
3. Click **Add New → Project**.
4. Import your GitHub repository.
5. Keep the default settings.
6. Click **Deploy**.

Your website will be live in a few seconds.

## Contact Form

The contact form is front-end only by default. To receive messages, connect it with:

* Formspree
* Netlify Forms
* Custom Backend API
